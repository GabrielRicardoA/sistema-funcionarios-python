# 💼 Sistema de Gerenciamento de Funcionários

Projeto desenvolvido em Python utilizando Programação Orientada a Objetos (POO), com persistência de dados em JSON.

---

## 🚀 Funcionalidades

* Adicionar funcionário CLT
* Adicionar funcionário Freelancer
* Listar funcionários
* Remover funcionário
* Calcular folha salarial total
* Salvar e carregar dados automaticamente com JSON

---

## 🧠 Conceitos aplicados

* Programação Orientada a Objetos (POO)

  * Abstração
  * Herança
  * Polimorfismo
* Manipulação de arquivos
* Estruturas de dados (listas e dicionários)
* Tratamento de erros (try/except)

---

## 🛠️ Tecnologias

* Python 3
* JSON

---

## ▶️ Como executar

```bash
python main.py
```
